Form Saver Chrome Extension

About

Form Saver is a Chrome extension that automatically saves your form input as you type. Never lose your form data again!

Features

  • Automatically saves form data as you type
  • Supports text, checkboxes, dates, and images
  • Keeps the last 15 entries
  • Works offline
  • Easy to use popup interface

Installation

Install Form Saver from the Chrome Web Store
